Objects
Effect
Determines how strongly this object affects every other square on the board when creating board interactivity net. The effect weakens with distance: first neighbours receive Suitability × Effect, second neighbours receive Suitability × Effect², and so on. Interactivity net is then used to simulate visitors movement during event.
Absorption
Describes how much of effect of other squares transmitted through this one would be absorbed here. I.e. observed square is [x,y], it's effect is 0.5 and suitability [20, 12, -20]. If square [x+1, y] has an object with absorption factor 0.8, that square in interactivity net would still be affected by 0.5 effect factor from square [x,y], however it'd keep 80% (0.8 abs factor) of effect that is heading towards [x+2,y] - meaning the multiplier would be 0.5*0.5*0.8
Suitability
How well this object appeals to each visitor demographic. Higher values raise that group's satisfaction score.
